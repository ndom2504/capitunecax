# Logement et installation

## Objectif
Donner à l’utilisateur une estimation réaliste du coût de l’installation.

## Éléments à couvrir
- Recherche de logement
- Dépôt de garantie si applicable
- Premier mois de loyer
- Mobilier de base
- Téléphone
- Internet
- Transport local
- Épicerie
- Assurance

## Réponse type
Si l’utilisateur demande un logement :
1. demander la ville
2. demander le budget
3. demander le nombre de personnes
4. proposer une fourchette de coût
5. rappeler qu’il faut vérifier les offres directement sur les sites officiels ou partenaires
