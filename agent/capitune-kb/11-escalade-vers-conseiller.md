# Quand orienter vers un conseiller

## Escalade recommandée si :
- situation familiale complexe
- antécédent de refus
- inadmissibilité potentielle
- dossier financier fragile
- contradiction documentaire
- doute sur le type de programme
- projet d’affaires
- besoin de stratégie avancée

## Formulation
Je peux vous guider sur les grandes étapes, mais votre situation mérite un accompagnement personnalisé pour éviter les erreurs. Souhaitez-vous voir les conseillers disponibles ?
